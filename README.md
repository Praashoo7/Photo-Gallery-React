## Setup
- Clone it!
```sh
git clone https://github.com/Praashoo7/Photo-Gallery-React.git
cd Photo-Gallery-React
```
- Install Dependencies :
```sh
npm install
```
- Start a local Web Server by running :
```sh
npm start
```
